# US-states-socio-economic-landscape

Interactive map, bar, and line graph to compare income per capita by state in the United States. I also use a scatterplot to see the correlation between income per capita with education and employment levels. Check the shiny web app here [https://lj6jui-ruddy-gunawan.shinyapps.io/us-states-socio-economic-landscape/](https://lj6jui-ruddy-gunawan.shinyapps.io/us-states-socio-economic-landscape/)
